# VideoSDK Call — Real-time 1:1 Voice/Video Calling

A Flutter app for real-time 1-to-1 voice and video calling over WebRTC, with live
presence and in-call text chat, backed by a small Node.js Socket.IO signalling server.

> Status: in active development. See the feature list below for what is complete vs partial.
